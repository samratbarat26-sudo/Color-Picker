# 🎨 Color Picker

A simple and responsive Color Picker web application built using HTML, CSS, and JavaScript.

## ✨ Features
- Pick any color using the built-in color input
- Live color preview box
- Displays the selected HEX color code
- Responsive and clean user interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
